Introduction to Verilog - II

Verilog is a Hardware Description Language (HDL). It is widely used by engineers in the electronics and VLSI fields, and it is a very significant tool in the development of the electronics industry. It was developed in 1984 by Phil Moorby and Prabhu Goel. There are other HDLs also present (like VHDL), but Verilog is very popular and easy to learn. We can get a better grasp on hardware modelling in Verilog as compared to VHDL. Verilog can be called a richer language, it has so many features that you will get tired of counting!

When a Verilog code is written and compiled, the hardware module is described by the software in terms of RTL schematic. This is a netlist in which logic gates and other digital blocks are used to carry out the described function(s). In the netlist, you can easily see the input wires and the output wires. It gives us an estimate of the physical complexity of our circuit. If we want to simulate the circuit or the digital block, then we have to write a separate code for simulation which is similar to the main code. This is called a test bench, and we will study about it soon. This is somewhat analogous to the test cases for a C/C++ code. The main use of such an extra code is to give inputs to the main code block and see the outputs being produced by it. Once you describe your digital circuit in Verilog and test its outputs after running a simulation, you can also check interconnect delays and power consumption statistics.

You must have learned in C/C++ programming that the code is executed in the order in which the sentences are written. Things work almost in the same manner in Verilog, along with some special blocks that have non sequential runtimes. You can specify a block of code to be executed in the beginning or even at a particular time. You need not write the code in the same order as that of the execution when you use the apt keywords. We will learn more about keywords and execution in the coming articles.

The syntax of Verilog is C based and people who have knowledge of C/C++ can easily learn Verilog. Apart from the basic computer programming constructs available in a programming language, Verilog also has many interesting features exclusive to Digital Electronics and Design. If you are well versed with the concepts of Digital Electronics, you will have a smooth journey of learning Verilog. In case you feel that you need some revision, you can refer Digital Basics.
